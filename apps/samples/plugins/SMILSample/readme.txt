Sample Application: SMILSample

A. Overview
This sample application demonstrates loading the OSMF SMIL plugin and using an external SMIL
document to ...


B. Installation Instructions (Flex Builder)

1. Unzip/copy the SMILSample project into your Flex Builder workspace folder. 
2. In Flex Builder, go to the File menu and select "Import".
3. Browse to your Flex Builder workspace folder.
4. Click "Finish". This will import the project.
5. Build the project.
6. Launch the application from the Run menu.

C. Usage Instructions

Run the app, you should see the captions appear.  Seeking should always show the correct caption at the correct time.

* IMPORTANT NOTE: The sample app loads the SMIL plugin statically. Due to Flash Player security, you will not be able to load 
the plugin dynamically unless you run the sample app from a Web server or localhost, i.e., "http://localhost/SMILSample.html".
Both the plugin and the player loading the plugin must be run from a Web server or localhost.
