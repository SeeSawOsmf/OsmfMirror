Multicast Player

A. Overview

The MulticastPlayer application demonstrates how to use the multicast feature of OSMF. It provides samples of how to subscribe to a multicast stream by using 

1) An F4M file
2) Stream specific information including connection string, RTMFP Groupspec and RTMFP StreamName

In order to run this application, Flash Player 10.1 or later is required. 

B. Installation Instructions (Flex Builder)

1. Unzip/copy the TransitionsSample project into your Flex Builder workspace folder. 
2. In Flex Builder, go to the File menu and select "Import".
3. Select "General", then "Existing Projects into Workspace", and click "Next".
4. Choose "Select root directory" and "Browse".
5. Browse to your Flex Builder workspace folder.
6. Select the checkbox next to "MulticastPlayer", and click "Finish".  This will import the project.
7. Build the project.
8. Launch the application from the Run menu.
